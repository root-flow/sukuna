## SUKUNA — Multi-Language Penetration Testing Tool

> **Python + C + Assembly** in a single file.  
> **OWASP Top 10**, **RCE Proof**, **GUI & CLI**.  
> **Author:** [`canmitm`](https://github.com/canmitm) | **Instagram:** [`@canmitm`](https://instagram.com/canmitm)

---

### Features

| Feature | Description |
|--------|-------------|
| **3-Language Engine** | Embedded C & Assembly modules (x3 each) |
| **Full OWASP Top 10** | SQLi, XSS, RCE, SSRF, LFI, Command Injection |
| **Advanced Scanning** | Nmap, SQLMap, XSSer, Commix, Nikto, Nuclei |
| **RCE Evidence** | Auto-saved payloads → `/var/lib/sukuna/evidence/` |
| **GUI + CLI** | Tkinter GUI with live logging |
| **One-Click Alias** | `sukuna` → runs anywhere (bash/zsh) |
